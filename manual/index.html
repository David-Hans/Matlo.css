Framework CSS para el diseño gráfico de objetos de aprendizaje

Bienvenido!

Y muchas gracias por usar Matlo.css, este manual te guiará en el correcto uso del framework. Para cada componente se dará una breve explicación de lo que éste hace, cómo funciona y se entregará el código necesario para que tú puedas hacer lo mismo.

Nota: algunos de los ejemplos ilustrados en este manual tienen código CSS extra para mejorar su visualización. El código presentado pretende demostrar la estructura de los componentes y se mantiene lo más simple posible para facilitar su comprensión.

Puedes navegar por todo el contenido usando la barra de navegación a la izquierda, pero te recomendamos partir con alguna de las siguientes opciones.

Comenzar

Para emperzar a trabajar inmediatamente.

About

Matlo.css fue diseñado siguiendo los principios de Material Design y desarrollado usando Materialize.

Para la creación de este Manual se utilizó Prettify.

  • Bienvenida
Matlo.css

Matlo.css (Material designed Learning Objects CSS) fue creado - usando Materialize - para el diseño gráfico de Objetos de Aprendizaje, como parte de un Trabajo de Titulación del Departamento de Ingeniería Informática de la Universidad de Santiago de Chile.
