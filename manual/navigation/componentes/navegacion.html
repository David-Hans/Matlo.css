La barra de navegación debe ser posicionada en el <header> de tus páginas, dentro de un nav tag y un contenedor div.
La barra tiene dos partes principales: el brand-logo y los enlaces de navegación.

Enlaces a la derecha

Agrega right al <ul> que contiene los enlaces.

Logo
  • CSS
  • Componentes
  • JavaScript
<nav>
    <div class="nav-wrapper">
        <a href="#!" class="brand-logo">Logo</a>
        <ul id="nav-mobile" class="right hide-on-med-and-down">
            <li><a href="#!">CSS</a></li>
            <li><a href="#!">Componentes</a></li>
            <li><a href="#!">JavaScript</a></li>
        </ul>
    </div>
</nav>

Enlaces a la izquierda

Agrega left al <ul> que contiene los enlaces.

  • CSS
  • Componentes
  • JavaScript
<nav>
    <div class="nav-wrapper">
        <a href="#!" class="brand-logo right">Logo</a>
        <ul id="nav-mobile" class="left">
            <li><a href="#!">CSS</a></li>
            <li><a href="#!">Componentes</a></li>
            <li><a href="#!">JavaScript</a></li>
        </ul>
    </div>
</nav>

Centrar el logo

Por defecto, el logo se centra en pantallas de tablets y móviles. Si quieres que esté siempre sentrado, agrega center a tu <a class="brand-logo">.

Logo
  • CSS
  • JavaScript
<nav>
    <div class="nav-wrapper">
        <a href="#!" class="brand-logo center">Logo</a>
        <ul id="nav-mobile" class="left">
            <li><a href="#!">CSS</a></li>
            <li><a href="#!">JavaScript</a></li>
        </ul>
    </div>
</nav>

Items activos

Agrega la clase active a tus tags <li> para indicar la página actual.

Logo
  • CSS
  • Componentes
  • JavaScript
<nav>
    <div class="nav-wrapper">
        <a href="#!" class="brand-logo left">Logo</a>
        <ul id="nav-mobile" class="right">
            <li><a href="#!">CSS</a></li>
            <li class="active"><a href="#!">Componentes</a></li>
            <li><a href="#!">JavaScript</a></li>
        </ul>
    </div>
</nav>

Barra fija

Para fijar la barra tienes que agregar otro div contenedor, con la clase navbar-fixed. Esto empujará el resto de tu contenido.

<div class="navbar-fixed">
    <nav>
        <div class="nav-wrapper">
            <a href="#!" class="brand-logo">Logo</a>
            <ul class="right hide-on-med-and-down">
                <li><a href="sass.html">Sass</a></li>
                <li><a href="badges.html">Components</a></li>
            </ul>
        </div>
    </nav>
</div>

Menú dropdown

Agrega la estructura de dropdown <ul> a la página, luego, agrega un elemento que active el menú dropdown; para esto, entrega el id de la estructura al atributo data-activates del activador.

  • Opción 2
  • Otra opción

<ul id="dropdown1" class="dropdown-content">
    <li><a href="#!">Opción 1</a></li>
    <li><a href="#!">Opción 2</a></li>
    <li class="divider"></li>
    <li><a href="#!">Otra opción</a></li>
</ul>
<nav>
    <div class="nav-wrapper">
        <a href="#!" class="brand-logo left">Logo</a>
        <ul class="right">
            <li><a href="#!">CSS</a></li>
            <li><a href="#!">Componentes</a></li>
            <!-- Dropdown Trigger -->
            <li><a class="dropdown-button" href="#!" data-activates="dropdown1">Dropdown<i class="material-icons right">arrow_drop_down</i></a></li>
        </ul>
    </div>
</nav>

Barra de búsqueda

Puedes agregar un form de búsqueda a la barra de navegación.

close 
<nav>
    <div class="nav-wrapper">
        <form>
            <div class="input-field">
                <input id="search" type="search" required>
                <label for="search"><i class="material-icons">search</i></label>
                <i class="material-icons">close</i>
            </div>
        </form>
    </div>
</nav>
