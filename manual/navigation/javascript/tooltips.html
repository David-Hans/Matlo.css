Es texto pequeño e interactivo usado generalmente para elementos gráficos, cuando uses íconos para acciones puedes usar un tooltip para clarificar su función

Básico

Agrega la clase tooltipped a tu elemento, luego puedes especificar detalles con data-position, data-delay o data-tooltip.

Arriba Abajo Izquierda Derecha 
<a class="btn tooltipped" data-position="top" data-delay="50" data-tooltip="Mensaje tooltip">Arriba</a>
<a class="btn tooltipped" data-position="bottom" data-delay="50" data-tooltip="Mensaje tooltip">Abajo</a>
<a class="btn tooltipped" data-position="left" data-delay="50" data-tooltip="Mensaje tooltip">Izquierda</a>
<a class="btn tooltipped" data-position="right" data-delay="50" data-tooltip="Mensaje tooltip">Derecha</a>

Inicialización

Los tooltips se inician automáticamente, pero, si los agregas dinámicamente, tienes que iniciarlos.

$(document).ready(function(){
    $('.tooltipped').tooltip({delay: 50});
});

Opciones JQuery

Nombre Descripción
delay El tiempo antes de que el tooltip aparezca (en milisegundos). (Default: 350)
tooltip Texto del tooltip. Puedes usar HTML si activas la opción html.
position Dirección del tooltip. 'top','right','bottom','left'. (Default: 'bottom')
html Permitir html dentro del tooltip. (Default: false)
