Puedes crear cajas de diálogo, mensajes de confirmación, u otro contenido usando modal.

Básico

Para que funcione, tienes que agregar el id del modal al link del activador.
Para agregar un botón de cerrar, agrega la clase modal-close a tu botón.

Cerrar 
<!-- Activador -->
<a class="waves-effect waves-light btn" href="#modal1">Modal</a>

<div id="modal1" class="modal">
    <div class="modal-content">
        <h4>Header del modal</h4>
        <p>Contenido</p>
    </div>
    <div class="modal-footer">
        <a href="#!" class=" modal-action modal-close btn-flat">Cerrar</a>
    </div>
</div>

Inicialización

Para poder abrir el modal, hay que inicializarlo.

$(document).ready(function(){
    $('.modal').modal();
});

Pie de página

Si tienes contenido muy largo y quieres que los botones de acción se mantengan visibles en todo momento, puedes agregar la clase modal-fixed-footer al modal.

Botón

Para usar modales con botones, tienes que especificar el id del modal en data-target, en vez de usar el atributo href.

<button data-target="modal3" class="btn">Modal</button>
