El carrusel es un componente robusto y versátil, puede ser una imagen, un slider o un carrusel de ítems.

Inicialización

Requiere incialización JQuery.

$(document).ready(function(){
    $('.carousel').carousel();
});

Opciones JQuery

Nombre Descripción
time_constant Tiempo de transición. (Default: 200)
dist Zoom de perspectiva. Si es 0, todos los items son del mismo tamaño. (Default: -100)
shift Espacio del item central. (Default: 0)
padding Padding entre items no-céntricos. (Default: 0)
full_width Hacer al carrusel un slider (como la siguiente sección).(Default: false)
indicators Mostrar indicadores. (Default: false)
no_wrap No dar la vuelta, al llegar al final. (Default: false)

Métodos JQuery

Ofrecemos los siguientes métodos para manipular el carrusel.

// Siguiente slide
$('.carousel').carousel('next');
$('.carousel').carousel('next', 3); // Avanzar n veces.
// Slide anterior
$('.carousel').carousel('prev');
$('.carousel').carousel('prev', 4); // Retroceder n veces.
// Ir al slide n
$('.carousel').carousel('set', 4);

Paneles

El carrusel no sólo permite el uso imágenes, también puede incluir contenido.
Puedes agregar ítems fijos al carrusel con la clase carousel-fixed-item.
