Usamos el sistema Grid responsivo de 12 columnas (estándar). Este Grid permite organizar tus diseños de forma fácil y ordenada.

Container

El container no es estrictamente parte del sistema Grid, pero, sirve para encapsular el contenido de tus páginas. Para usar un container tienes que declarar la clase container dentro de un <div>, como se muestra a continuación.

<body>
    <div class="container">
        <!-- Contentenido -->
    </div>
</body>

Nuestro container es de ancho completo, por lo que ocupa todo el espacio disponible

Offset

Permite acomodar o empujar columnas, para usarlo hay que agregar la clase offset-s2, donde s es el tamaño de pantalla (Diseño responsivo) y 2 es el número de columnas del offset.

12 columnas (0 offset)
10 columnas (2 offset)
6 columnas (6 offset)
4 col (2 offset)
4 col (2 offset)
<div class="row">
    <div class="col s12">12 columnas (0 offset)</div>
    <div class="col s10 offset-s2">10 columnas (2 offset)</div>
    <div class="col s6 offset-s6">6 columnas (6 offset)</div>
    <div class="col s4 offset-s2">4 col (2 offset)</div>
    <div class="col s4 offset-s2">4 col (2 offset)</div>
</div>

Diseño Responsivo

Con el diseño responsivo, puedes crear páginas que se ven bien en cualquier dispositivo

Uso

En los ejemplos anteriores sólo definimos el tamaño para pantallas pequeñas, usando "col s12". Esto está bien si queremos un diseño fijo, ya que las reglas se propagan hacia arriba. Al usar s12, escencialmente estamos diciendo "col s12 m12 l12", pero, al definir explícitamente los tamaños podemos crear sitios responsivos.

Cambia el tamaño de tu ventana de navegación para ver las diferencias en los ejemplos a continuación
