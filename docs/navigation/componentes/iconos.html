Incluimos más de 900 íconos, cortesía de Google.

Uso

Para usar los íconos, el head de tu código HTML debe contener la siguiente línea.

<link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">

Ahora, puedes usar cualquiera de los íconos disponibles.

get_app
near_me
sync_disabled
power_settings_new
<div class="row">
    <div class="col s3 center-align"><i class="material-icons">get_app</i></div>
    <div class="col s3 center-align"><i class="material-icons">near_me</i></div>
    <div class="col s3 center-align"><i class="material-icons">sync_disabled</i></div>
    <div class="col s3 center-align"><i class="material-icons">power_settings_new</i></div>
</div>

Tamaños

Para controlar el tamaño de los íconos, puedes modificar la propiedad font-size. Opcionalmente, puedes usar las clases que ofrecemos.

music_note
music_note
music_note
music_note
tiny
small
medium
large
<div class="row">
    <div class="col s3 center-align"><i class="tiny material-icons">music_note</i></div>
    <div class="col s3 center-align"><i class="small material-icons">music_note</i></div>
    <div class="col s3 center-align"><i class="medium material-icons">music_note</i></div>
    <div class="col s3 center-align"><i class="large material-icons">music_note</i></div>
</div>
