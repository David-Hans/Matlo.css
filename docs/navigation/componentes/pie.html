Los footers son una gran forma de organizar la navegación del sitio e información, al final de la página. Es el lugar donde el usuario se encontrará al terminar de interactuar con el contenido principal.

Básico

Nota: Este framework usa flexbox para estructurar el html y lograr que el footer esté siempre al final de la página. Es importante mantener toda la estructura de tus páginas dentro de los tres HTML5 tags: <header>, <main> y <footer>

Footer

Puedes usar row y col, para organizar el contenido de tu footer.
