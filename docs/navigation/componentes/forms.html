Input field

Para los campos de texto, debes tener un input-field div alrededor de tus input y label, así podemos animarlos con JQuery. Esto sólo se aplica a los input y textarea de las form.

La clase validate usa validación de HTML5 y agrega la clase valid o invalid acordemente. Si no quieres la validación verde o roja, remueve la clase validate de tus inputs.

Input en linea: 
<div class="row">
    <form class="col s12">
        <div class="row">
            <div class="input-field col s6">
                <input placeholder="Placeholder" id="first_name" type="text" class="validate">
                <label for="first_name">Nombre</label>
            </div>

            <div class="input-field col s6">
                <input id="last_name" type="text" class="validate">
                <label for="last_name">Apellido</label>
            </div>
        </div>

        <div class="row">
            <div class="input-field col s12">
                <input disabled value="No se puede escribir aquí" id="disabled" type="text" class="validate">
                <label for="disabled">Deshabilitado</label>
            </div>
        </div>
        <div class="row">
            <div class="input-field col s12">
                <input id="password" type="password" class="validate">
                <label for="password">Contraseña</label>
            </div>
        </div>

        <div class="row">
            <div class="input-field col s12">
                <input id="email" type="email" class="validate">
                <label for="email">Email</label>
            </div>
        </div>

        <div class="row">
            <div class="col s12">
                Input en línea:
                <div class="input-field inline">
                    <input id="email" type="email" class="validate">
                    <label for="email" data-error="Ingrese un email válido" data-success="Gracias!">Email</label>
                </div>
            </div>
        </div>
    </form>
</div>

Nota: Con data data-error y data-success puedes agregar mensajes personalizados de validación (como se ve en el input en línea de Email).


Íconos

Puedes agregar íconos a tus inputs. Sólo tienes que agregar el ícono con la clase prefix, antes del input y el label.

account_circle
phone 
<div class="row">
    <form class="col s12">
        <div class="row">
            <div class="input-field col s6">
                <i class="material-icons prefix">account_circle</i>
                <input id="icon_prefix" type="text" class="validate">
                <label for="icon_prefix">Nombre</label>
            </div>

            <div class="input-field col s6">
                <i class="material-icons prefix">phone</i>
                <input id="icon_telephone" type="tel" class="validate">
                <label for="icon_telephone">Teléfono</label>
            </div>
        </div>
    </form>
</div>

Textarea

Permiten al usuario el ingreso de texto más grande y ajustable. Debes tener un input-field div alrededor de tus input y label.

Las Textareas ajustan su tamaño según el texto que contienen.

<div class="row">
    <form class="col s12">
        <div class="row">
            <div class="input-field col s12">
                <textarea id="textarea1" class="materialize-textarea"></textarea>
                <label for="textarea1">Textarea</label>
            </div>
        </div>
    </form>
</div>

Íconos

También se pueden agregar íconos.

        <div class="row">

                <textarea id="icon_prefix2" class="materialize-textarea"></textarea>
                <label for="icon_prefix2">First Name</label>
            </div>
        </div>
    </form>
</div>

Select

Debe estar rodeado por un input-field. Y es un plugin JQuery, así que debe ser inicializado de la siguente forma.

$(document).ready(function(){
    $('select').material_select();
});

Tenemos selects estándar, múltiples (multiple) y con opciones agrupadas (optgroup).

<div class="row">
    <div class="input-field col s12 m4">
        <select>
            <option value="" disabled selected>Elige tu opción</option>
            <option value="1">Opción 1</option>
            <option value="2">Opción 2</option>
            <option value="3">Opción 3</option>
        </select>
        <label>Estándar</label>
    </div>

    <div class="input-field col s12 m4">
        <select multiple>
            <option value="" disabled selected>Elige tu opción</option>
            <option value="1">Opción 1</option>
            <option value="2">Opción 2</option>
            <option value="3">Opción 3</option>
        </select>
        <label>Múltiple</label>
    </div>

    <div class="input-field col s12 m4">
        <select>
            <option value="" disabled selected>Elige tu opción</option>
            <optgroup label="team 1">
                <option value="1">Opción 1</option>
                <option value="2">Opción 2</option>
                </optgroup>
            <optgroup label="team 2">
                <option value="3">Opción 3</option>
                <option value="4">Opción 4</option>
            </optgroup>
        </select>
        <label>Grupos</label>
    </div>
</div>

Imágenes

Puedes agregar imágenes a tus opciones en cualquier tipo de select. En la option agregas la fuente de la imágen, con el atributo data-icon. También, puedes agregar las clases left o right para alinear los íconos, o circle para redondearlos.

<div class="input-field col s12 m6">
    <select class="icons">
        <option value="" disabled selected>Elige tu opción</option>
        <option value="" data-icon="https://unsplash.it/49/49.jpg" class="circle">Ejemplo 1</option>
        <option value="" data-icon="https://unsplash.it/49/51.jpg" class="circle">Ejemplo 2</option>
        <option value="" data-icon="https://unsplash.it/49/50.jpg" class="circle">Ejemplo 1</option>
    </select>
    <label>Select con imágenes</label>
</div>

<div class="input-field col s12 m6">
    <select class="icons">
        <option value="" disabled selected>Elige tu opción</option>
        <option value="" data-icon="https://unsplash.it/50/49.jpg" class="left circle">Ejemplo 1</option>
        <option value="" data-icon="https://unsplash.it/51/49.jpg" class="left circle">Ejemplo 2</option>
        <option value="" data-icon="https://unsplash.it/50/50.jpg" class="left circle">Ejemplo 3</option>
    </select>
    <label>Select con imágenes</label>
</div>

Deshabilitado

Puedes agregar disabled al elemento select para deshabilitarlo completamente.

<label>Select Deshabilitado</label>
<select disabled>
    <option value="" disabled selected>Elige tu opción</option>
    <option value="1">Opción 1</option>
    <option value="2">Opción 2</option>
    <option value="3">Opción 3</option>
</select>

Radio Buttons

Para agregar, tienes que tener el mismo atributo name para cada botón en el grupo. Se puede usar el atributo disabled.

<form action="#">
    <p>
        <input name="group1" type="radio" id="test1" />
        <label for="test1">Opción 1</label>
    </p>
    <p>
        <input name="group1" type="radio" id="test2" />
        <label for="test2">Opción 2</label>
    </p>
    <p>
        <input class="with-gap" name="group1" type="radio" id="test3"  />
        <label for="test3">Opción 3</label>
    </p>
    <p>
        <input name="group1" type="radio" id="test4" disabled="disabled" />
        <label for="test4">Opción 4</label>
    </p>
</form>

Puedes crear radio buttons con borde interior, usando la clase with-gap (como la opción 3, del ejemplo anterior).
