Hay tres tipos de botones principales descritos en Material Design. El botón elevado implica acciones y le da relieve a una página principalmente plana; el flotante circular es para acciones muy importantes; y los botones planos son usualmente para elementos que ya tienen profundidad, como Cartas o Modales.

Plano

Los botones planos son usado para reducir el relieve excesivo.

BOTÓN 
<a class="waves-effect waves-red btn-flat">BOTÓN</a>

Enviar

Cuando uses botones para enviar un form, usa un botón con type="submit".

<button class="btn waves-effect waves-light" type="submit" name="action">
    enviar
    <i class="material-icons right">send</i>
</button>

Grande

Para botones que necesitan más atención usa btn-large.
