Básico

Puedes agregar una lista dropdown a cualquier botón.
Debes asegurarte de que el atributo data-activates del activador calce con el id del tag <ul> del dropdown.

arrow_drop_down Dropdown
  • Opción 1
  • Opción 2
  • Opción 3
<a class='dropdown-button btn' href='#' data-activates='dropdown1'>
    <i class="material-icons right">arrow_drop_down</i>
    Dropdown
</a>
<ul id='dropdown1' class='dropdown-content'>
    <li><a href="#!">Opción 1</a></li>
    <li><a href="#!">Opción 2</a></li>
    <li class="divider"></li>
    <li><a href="#!">Opción 3</a></li>
</ul>

Inicialización

$('.dropdown-button').dropdown();

Opciones JQuery

Nombre Descripción
induration Duración de la transición de entrada en milisegundos. (Default: 300)
outduration Duración de la transición de salida en milisegundos. (Default: 225)
constrainwidth Restringir el ancho del dropdown, según el tamaño del activador. (Default: true)
hover Abrir el dropdown al pasar el puntero por el activador. (Default: false)
gutter Espacio desde el borde. (Default: 0)
beloworigin Mostrar el dropdown bajo el activador. (Default: false)
alignment Borde al que se alinea el menú. (Default: left)

Métodos JQuery

Ofrecemos los siguientes métodos para manipular el dropdown.

// Abrir el dropdown
$('.dropdown-button').dropdown('open');
// Cerrar el dropdown
$('.dropdown-button').dropdown('close');
