Waves es una libraría que permite crear el efecto de tinta de Material Design.

Básico

Para aplicar Waves, sólo tienes que agregar la clase waves-effect, también puedes agregar color al efecto.

add Send Send 
<a href="#!" class="btn-floating btn-large waves-effect waves-light">
    <i class="material-icons">add</i>
</a>
<a href="#!" class="btn waves-effect orange">Send</a>
<a href="#!" class="btn waves-effect waves-orange">Send</a>

Personalizar

Puedes hacer tus propios colores, con un poco de CSS.

/*
Cuando crees tu CSS, cambia "brown"
y el valor rgba, por el color que
desees (manteniendo el alpha)
*/
.waves-effect.waves-brown .waves-ripple {
    /* El valor alpha permite que el texto
    y fondo del botón se puedan ver. */
    background-color: rgba(121, 85, 72, 0.65);
}

Círculo

Si quieres que Waves forme círculos, puedes agregar la clase waves-circle.
