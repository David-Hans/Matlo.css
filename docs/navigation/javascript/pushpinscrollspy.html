Pushpin

Es el plugin de posición fija que proveemos, lo puedes usar para dejar elementos de tu página estáticos, durante rangos específicos de avance.

Hay un ejemplo en vivo en esta página: la tabla de contenidos a la derecha, que se queda con el usuario a medida que éste se mueve por el contenido.


Inicialización

$(document).ready(function(){
    $('.target').pushpin({
        top: 0,
        bottom: 1000,
        offset: 0
    });
});

Nota: reemplazar target con la clase del elemento que se desea fijar.


Opciones JQuery

Nombre Descripción
top Distancia en pixeles desde la parte superior de la página, hasta el punto en que el elemento se queda fijo. (Default: 0)
bottom Distancia en pixeles desde la parte superior de la página, hasta el punto en que el elemento deja de quedar fijo. (Default: infinito)
offset El offset desde la parte superior de la página en que el elemento estará fijo. (Default: 0)

Scrollspy

Plugin de JQuery que trackea ciertos elementos y la posición actual de la pantalla, para realizar cambios acordemente.

La tabla de contenidos a la derecha también es un ejemplo de esto, como puedes notar, el elemento activo de la tabla cambia a medida que se avanza en el contenido.


Estructura

El contenido principal de esta página (contenido + tabla de contenidos) funciona de la siguiente forma.

<div class="row">
    <div class="col s12 m9 l10">
        <div id="id1" class="section scrollspy">
            <p>Contenido 1</p>
        </div>

        <div id="id2" class="section scrollspy">
            <p>Contenido 2</p>
        </div>

        <div id="id3" class="section scrollspy">
            <p>Contenido 3</p>
        </div>
    </div>

    <div class="col hide-on-small-only m3 l2">
        <div class="toc-wrapper">
            <ul class="section table-of-contents">
                <li><a href="#id1">Contenido 1</a></li>
                <li><a href="#id2">Contenido 2</a></li>
                <li><a href="#id3">Contenido 3</a></li>
            </ul>
        </div>
    </div>
</div>

Nota 1: para que Scrollspy funcione es importante que los href de los elementos de la tabla de contenidos sean iguales a los id de cada div de tu contenido.

Nota 2: para que Pushpin funcione, en este ejemplo específico, tiene que ser inicializado de la siguiente forma (fijando la tabla de contenidos).

$(document).ready(function(){
    $('.toc-wrapper').pushpin({
        (...)
    });
});

Inicialización

$(document).ready(function(){
    $('.scrollspy').scrollSpy();
});
