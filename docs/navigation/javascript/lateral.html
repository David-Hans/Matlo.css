Inicialización

$(".button-collapse").sideNav();

Opciones

Nombre Descripción
menuWidth Ancho de la barra. (Default: 300)
edge Origen horizontal (izquierda o derecha). (Default: 'left')
closeOnClick Cierra la barra al hacer click en un <a>. (Default: false)
draggable Si se puede arrastrar para abrir rel menú (touch-screen). (Default: true)

Métodos JQuery

Ofrecemos los siguientes métodos para manipular la barra lateral.

// Mostrar barra
  $('.button-collapse').sideNav('show');
  // Esconder barra
  $('.button-collapse').sideNav('hide');
  // Destruir barra
  $('.button-collapse').sideNav('destroy');

Fija

Puedes agregar la clase fixed para que la barra quede abierta y fija en pantallas de escritorio; y se esconda en pantallas más pequeñas. La barra de navegación lateral de esta página (a la izquierda) es un ejemplo de esto.

<ul id="slide-out" class="side-nav fixed">
    <li><a href="#!">Enlace 1</a></li>
    <li><a href="#!">Enlace 2</a></li>
</ul>
<a href="#" data-activates="slide-out" class="button-collapse">
    <i class="material-icons">menu</i>
</a>

Si quieres usar este tipo de barra, tendras que empujar tu contenido principal una cantidad de pixeles igual a el ancho de la barra, para pantallas de escritorio. A continuación, un ejemplo de cómo hacer esto con CSS.

header, main, footer{
    padding-left: 300px;
}

@media only screen and (max-width : 992px) {
    header, main, footer {
        padding-left: 0;
    }
}
